# Projet-Incident-Grand-Lome

Projet Power BI – Analyse des incidents dans le Grand Lomé
**Contexte

Ce projet analyse les incidents signalés dans le Grand Lomé (Togo), incluant les appels d’urgence, les interventions effectuées et leurs conclusions, afin d’évaluer la performance opérationnelle des services d’intervention.

**Objectifs

Identifier les types d’incidents les plus fréquents
Analyser les temps de réponse des unités d’intervention
Localiser les zones les plus touchées (quartiers & communes)
Évaluer les moyens déployés selon la gravité des incidents

**Données utilisées

Les données contiennent notamment :
Numéro et date d’incident
Type d’appel (112, 117, initiative agent)
Quartier et commune
Type d’incident
Indicateur de santé mentale
Nombre d’unités déployées
Temps de réponse (en minutes)
Conclusion de l’appel

** Nettoyage effectué : correction des problèmes d’encodage (ex : caractères spéciaux), standardisation des dates, heures et catégories.

🛠️ Outils

Microsoft Power BI : modélisation, DAX, visualisations
GitHub : documentation et versioning

📈 Indicateurs clés (KPIs)

Nombre total d’incidents
Temps de réponse moyen
Incidents par type
Incidents par commune
Répartition des conclusions d’intervention

🔍 Résultats & insights

Certains quartiers concentrent une forte fréquence d’incidents
Les accidents moto présentent des temps de réponse plus élevés
Les violences conjugales sont majoritairement suivies de transferts ou arrestations
